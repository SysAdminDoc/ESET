# ESET Port & Address Reference

![License](https://img.shields.io/badge/license-MIT-blue)
![Type](https://img.shields.io/badge/type-Reference-informational)

Complete ESET endpoint security port and address reference lists for firewall configuration and network policy management. Essential for sysadmins deploying ESET in managed environments.

## Contents

Comprehensive lists of all ports and addresses required for:

- **ESET Endpoint Security** — Client-to-server communication
- **ESET PROTECT Console** — Management server connectivity
- **Update Servers** — Virus definition and module update endpoints
- **Cloud Services** — ESET LiveGrid, EDTD, and cloud-based detection
- **License Activation** — Activation and licensing server endpoints

## Usage

Use these lists to configure firewall rules, proxy allowlists, or network policies in your environment. Import directly into your firewall management tool or reference during ESET deployment.

## Source

Compiled from official ESET documentation and verified against production environments.
